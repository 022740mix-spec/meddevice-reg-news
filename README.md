# MedDevice Reg News — 医療機器規制ニュース

医療機器の法規制・ガイダンス・規格・承認・リコール・サイバーセキュリティの最新ニュースを日本語で発信するサイトです。

## カテゴリ

| カテゴリ | 対象 |
|---------|------|
| 特集 | 特集記事・深掘り解説 |
| 法規制 | 薬機法、FD&C Act、MDR/IVDR |
| ガイダンス | FDA Guidance、PMDA 通知、Q&A |
| 規格 | ISO 13485、IEC 62304、ISO 14971 等 |
| 承認・認証 | 510(k)、PMA、CE マーキング、PMDA 承認 |
| リコール・安全 | FDA recall、PMDA 回収、安全性情報 |
| サイバーセキュリティ | 医療機器セキュリティ、SBOM |
| 海外動向 | 各国規制当局の動き |

## 技術スタック

- React 19 + Vite 7
- GitHub Pages でホスティング
- Atom フィード自動生成

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動
npm run build    # 本番ビルド
npm run deploy   # GitHub Pages にデプロイ
```

## ベース

[AI-News](https://github.com/022740mix-spec/AI-News) のサイトテンプレートを医療機器規制向けに転用。
